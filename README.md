# 🧠 CIFAR-10 Image Classification with CNN

## 📌 Project Description

This project focuses on building a **Convolutional Neural Network (CNN)** to classify images from the CIFAR-10 dataset.

The goal is to correctly predict the class of an image among 10 categories such as airplanes, cars, birds, cats, and more.

---

## 📊 Dataset

The project uses the :contentReference[oaicite:0]{index=0} dataset.

### Dataset Details:

- 60,000 color images (32×32 pixels)
- 10 classes:
  - Airplane
  - Automobile
  - Bird
  - Cat
  - Deer
  - Dog
  - Frog
  - Horse
  - Ship
  - Truck

- Split:
  - 50,000 training images
  - 10,000 test images

---

## ⚙️ Data Preprocessing

- Normalization of pixel values
- Data augmentation:
  - Random horizontal flip
  - Random crop with padding
- Conversion to tensors

---

## 🧠 Model Architecture

The CNN architecture consists of:

- 3 Convolutional layers
- ReLU activation
- MaxPooling layers
- Fully connected layers
